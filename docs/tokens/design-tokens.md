# Design Tokens

Design tokens are reusable design decisions that help maintain consistency across product, brand, and campaign experiences.

## Color Tokens

| Token | Value | Usage |
|---|---|---|
| `--color-brand-primary` | `#8F00FF` | Primary brand color |
| `--color-brand-light` | `#F4E8FF` | Light brand background |
| `--color-text-primary` | `#111827` | Main text |
| `--color-text-secondary` | `#6B7280` | Secondary text |
| `--color-border-default` | `#E5E7EB` | Default border |

## Typography Tokens

| Token | Value | Usage |
|---|---|---|
| `--font-size-xs` | `12px` | Caption |
| `--font-size-sm` | `14px` | Small text |
| `--font-size-md` | `16px` | Body text |
| `--font-size-lg` | `20px` | Section title |
| `--font-size-xl` | `32px` | Page title |

## Spacing Tokens

## Radius Tokens

| Token | Value | Usage |
|---|---|---|
| `--radius-sm` | `8px` | Small elements |
