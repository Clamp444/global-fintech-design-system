# Skills Library

Skills turn the design system into repeatable workflows for AI-assisted design, review, documentation, and production handoff.

## Core Skills

| Skill | Purpose |
|---|---|
| [PalmPay Design System](./palmpay-system-design.md) | Generate, review, and map PalmPay product UI using system rules. |
| [AI Generation Rules](./ai-generation-rules.md) | Define how AI should generate editable Figma drafts from the system. |
| [Prompt Templates](./ai-prompt-templates.md) | Provide reusable prompts for common PalmPay generation tasks. |
| [Component Inventory](./ai-component-inventory.md) | Map AI generation needs to reusable components and gaps. |
| [Figma Mapping](./figma-mapping.md) | Connect documentation rules to PalmPay Figma assets. |
